Divisi
Program

Divisi Program adalah divisi dalam ekskul IT yang memfokuskan pada pengembangan keterampilan pemrograman dasar menggunakan bahasa pemrograman C++, serta pembuatan halaman web dengan menggunakan HTML dan CSS. Dalam divisi ini, kamu akan mendapatkan kesempatan untuk memahami konsep dasar pemrograman, struktur data, dan algoritma melalui bahasa C++.

  • Dasar Pemrograman Dengan C++
    Dengan C++, kamu akan belajar bagaimana membuat program komputer sederhana, mengerti sintaks dasar, variabel, tipe data, pernyataan pengulangan (looping), percabangan (conditional), serta fungsi. Kamu juga akan diperkenalkan dengan konsep pengelolaan memori dan pemrograman berorientasi objek melalui C++.

  • HTML (Hypertext Markup Language)
    Divisi Program dalam ekskul IT mempelajari HTML, yang merupakan bahasa markup yang digunakan untuk membangun struktur dan elemen pada halaman web. Dalam divisi ini, kamu akan belajar tentang tag-tag HTML, seperti pembuatan heading, paragraf, gambar, link, tabel, dan formulir. Kamu akan menguji keterampilanmu dalam menggunakan HTML untuk membuat halaman web yang menarik dan fungsional.

  • CSS (Cascading Stylesheet)
    Divisi Program dalam ekskul IT fokus pada pengembangan keterampilan pemrograman, termasuk belajar tentang Cascading Style Sheets (CSS). Dalam divisi ini, kamu akan mempelajari penggunaan CSS untuk mengubah tampilan, warna, dan tata letak elemen-elemen pada halaman web.

Dengan mengikuti divisi program ini, kamu akan memperoleh dasar-dasar yang kuat dalam pemrograman dan pembuatan halaman web, yang akan membantumu dalam memahami konsep-konsep yang lebih kompleks di bidang IT dan mengembangkan keterampilanmu di masa depan.
